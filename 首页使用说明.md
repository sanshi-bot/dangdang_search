# 数据采集平台 - 首页使用说明

## 📋 概述

新增了一个统一的首页（`frontend/home.html`），将当前的当当网图书爬虫作为其中一个功能模块展示。

## 🎯 功能特点

### 1. 模块化设计
- 首页采用卡片式布局，每个功能模块独立展示
- 清晰的功能分类和描述
- 支持未来扩展更多功能模块

### 2. 后端状态检测
- **实时监控**：首页会自动检测后端服务是否在线
- **状态指示器**：右上角显示后端服务状态（运行中/已断开）
- **智能拦截**：后端未启动时，无法进入功能页面，会弹出提示
- **自动跳转**：在功能页面时，如果后端断开，3秒后自动返回首页

### 3. 当前可用模块
- **当当网图书爬虫**：点击即可跳转到原有的爬虫功能页面
  - 需要后端服务运行才能进入
  - 左上角有返回首页按钮

### 4. 预留模块（即将上线）
- 电商数据采集
- 新闻资讯采集
- 数据分析中心
- 任务管理
- 数据检索

## 🚀 使用方法

### 方式一：使用启动脚本（推荐）
```bash
# 启动全部服务（后端+前端首页）
start_all.bat

# 或者分别启动
start_backend.bat   # 启动后端
start_frontend.bat  # 启动前端首页
```

### 方式二：直接打开文件
直接在浏览器中打开 `frontend/home.html`

## 🔒 安全机制

### 1. 首页保护
- 后端未启动时，爬虫模块卡片会显示为禁用状态
- 点击时会弹出提示："后端服务未启动，请先启动后端服务！"
- 按钮文字变为"需要后端服务"

### 2. 功能页面保护
- 进入爬虫页面时，会立即检测后端状态
- 如果后端未启动，3秒后自动返回首页
- 在使用过程中，如果后端断开，也会在3秒后自动返回首页

### 3. 返回首页
- 爬虫页面左上角有"🏠 返回首页"按钮
- 点击即可返回首页
- 悬停时有动画效果

## 📁 文件结构

```
frontend/
├── home.html          # 首页HTML
├── home-app.js        # 首页JavaScript逻辑（含后端检测）
├── home-style.css     # 首页样式
├── index.html         # 爬虫功能页面（含返回按钮）
├── app.js            # 爬虫功能逻辑（含自动跳转）
└── style.css         # 爬虫功能样式（含返回按钮样式）
```

## 🎨 页面特点

1. **美观的渐变背景**：紫色渐变背景，视觉效果出色
2. **响应式设计**：自适应不同屏幕尺寸
3. **流畅动画**：卡片悬停效果和进入动画
4. **清晰的状态标识**：
   - 后端在线：绿色圆点，显示"后端服务运行中"
   - 后端离线：红色圆点，显示"后端服务已断开"
   - 检测中：黄色背景，显示"检测中..."

## 🔄 页面跳转流程

```
首页 (home.html)
  ↓ 点击爬虫卡片（需要后端在线）
爬虫页面 (index.html)
  ↓ 点击左上角返回按钮 或 后端断开自动跳转
首页 (home.html)
```

## 💡 使用场景

### 场景1：正常使用
1. 运行 `start_all.bat` 启动后端和前端
2. 首页显示后端在线（绿色圆点）
3. 点击"当当网图书爬虫"卡片进入功能页面
4. 使用完毕后点击左上角"返回首页"按钮

### 场景2：后端未启动
1. 直接打开 `home.html`
2. 首页显示后端离线（红色圆点）
3. 点击爬虫卡片会弹出提示
4. 需要先启动后端服务

### 场景3：使用中后端断开
1. 在爬虫页面使用功能
2. 后端服务意外关闭
3. 页面检测到断开，显示提示
4. 3秒后自动返回首页

## 🛠️ 技术栈

- Vue 3：前端框架
- Axios：HTTP请求库
- CSS3：样式和动画
- 原生JavaScript：交互逻辑

## 📝 注意事项

1. 首页会每15秒自动检测一次后端状态
2. 爬虫页面在后端断开后会自动返回首页
3. 预留模块目前不可点击，标记为"即将上线"
4. 返回首页按钮固定在左上角，不会随页面滚动
